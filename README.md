# HTML Basics Examples

This repository contains simple and beginner-friendly **HTML-only** code examples.  
All files include pure HTML without any CSS or JavaScript to help you understand the core structure and tags clearly.

---

##  What's Included

###  Basic HTML Structure
- HTML skeleton  
- Headings, paragraphs, line breaks  
- Comments  
- Formatting tags (bold, italic, underline, small)  

###  Text & Media
- Images  
- Audio  
- Video  
- Marquee text  
- Preformatted text  

###  Lists & Tables
- Ordered lists  
- Unordered lists  
- Different list types (A, a, I, i)  
- Tables with border  
- Tables with **rowspan** and **colspan**

###  Forms & Inputs
- Text, email, password input  
- Radio buttons  
- Checkboxes  
- Dropdown (select)  
- File input  
- Button examples  

###  Links
- Normal links  
- Email link (`mailto:`)  
- Phone link (`tel:`)  
- File download link  

###  Semantic HTML
- Header  
- Nav  
- Section  
- Footer  

###  Embeds
- Iframe  
- YouTube/video/audio examples  

---

##  How to Use

1. Copy any HTML code sample.
2. Paste it into a file named **index.html**.
3. Open it in any browser (Chrome, Edge, Firefox, etc.)
4. Learn by modifying the examples.

---

##  Requirements

No software installation is needed.  
A simple browser is enough to run all examples.
